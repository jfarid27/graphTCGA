What is this?

This is a repository for precomputed PANDA analysis. Use it to download the results in various formats to analyze with your favorite network science tools.

How do I use this?

  1. Search for a cancer dataset.
  2. Visualize a demo (Z-score > 5.5) by clicking "Visualize" in the results table.
  3. Choose a Z-Score exclusion range in histogram.
  4. Click "Download" in the results table to download Panda analyzed network.

Downloadable Network Science Tools

  • Gephi (file format supported)
  • Cytoscape (file format supported)
  • Pajek
  • SNA (R programming language package)
  • SNAP (Python programming language package)
